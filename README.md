# Estudos sobre IA

Portal pessoal para organizar:

- ferramentas de inteligência artificial;
- ideias de agentes e produtos;
- trilhas de estudo;
- casos de uso e limitações;
- ferramentas já utilizadas, em teste ou a estudar.

## Publicação

O projeto é uma página HTML estática e auto-contida.

O workflow em `.github/workflows/pages.yml` publica automaticamente no GitHub Pages a cada alteração na branch `main`.

## Estrutura

- `index.html`: aplicação completa.
- `.nojekyll`: impede o processamento pelo Jekyll.
- `.github/workflows/pages.yml`: implantação automática no GitHub Pages.
